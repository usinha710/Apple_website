🍎 Apple Homepage Clone

A responsive clone of the Apple homepage built using HTML and CSS to practice modern frontend development and UI design. The project recreates the layout and visual structure of Apple's official website, focusing on clean design and responsive behavior.

🚀 Features

Recreated the Apple homepage layout and UI structure

Fully responsive design for different screen sizes

Clean and organized HTML & CSS code

Includes navigation bar, product sections, and promotional banners

Designed to closely match the look and feel of Apple's official website

🛠️ Tech Stack

HTML5

CSS3
